アトリ 为一个在`Linux`下运行的QQBot

此分支实现方式是 `OPQBot + Python-iotbot` 以 `Python` 为主语言的项目

**前身框架 CoolQ**

**谨此纪念 CoolQ 的停运**

**“愿我们能在更开放的平行世界里相遇”**

此项目名称来源于

[ANIPLEX.EXE](https://aniplex-exe.com/) 发行作品 [ATRI -My Dear Moments-](https://atri-mdm.com/) 中的 [アトリ](https://atri-mdm.com/character/)
